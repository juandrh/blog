Condicionales 

def signo(x):
    if x > 0:
        print(x, "tiene signo positivo")
    elif x < 0:
        print(x, "tiene signo negativo")    
    else:
        print(x, "is cero")

signo(-1)
signo(0)
signo(1)

-1 tiene signo negativo
0 is cero
1 tiene signo positivo
